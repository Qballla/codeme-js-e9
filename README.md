### Operatory artmetyczne
| operator | opis |
| :---: | ---- |
| + | dodawanie |
| - | odejmowanie |
| * | mnożenie |
| / | dzielenie |
| % | reszta z dzielenia - modulo |
| == | równy |
| === | równa wartosc i ten sam typ |
| != | różny |
| !== | różna wartosc i różny typ |
| > | większy niż |
| < | mniejszy niż |
| >= | więszy równy |
| <= | mniejszy równy |

### Operatory logiczne
| operator | opis |
| :---: | ---- |
| && | logiczne AND |
| \|\| | logiczne OR |
| ! | logiczne NOT |
